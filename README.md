# Personal Website – Vishay Agarwal

This repository contains the source code for my personal website, designed to showcase my work in machine learning, research-oriented projects, and student-led initiatives.

The site highlights selected AI projects, leadership experience, and ways to connect, with a strong emphasis on clarity, hierarchy, and professional presentation for college admissions and internship audiences.

## Features

- Clean, responsive single-page layout
- Clear visual hierarchy with large, readable typography
- Smooth scrolling navigation
- Color-coded project cards for quick differentiation
- Distinct button systems for projects, leadership, and contact
- Fully static (HTML + CSS only, no frameworks)

## Sections

- **Hero** – Introduction, focus areas, and quick navigation
- **Selected Projects** – Machine learning and AI projects with GitHub links
- **Leadership & Activities** – Student-led nonprofits and organizations
- **Let’s Connect** – Email, GitHub, and resume access

## Tech Stack

- HTML5
- CSS3
- Google Fonts (Inter, Space Grotesk)
